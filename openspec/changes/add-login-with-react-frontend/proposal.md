# 新增登录功能（含 React 前端）

## 目标
为现有 Spring Boot 项目增加用户登录功能，提升系统安全性和用户体验。前端采用 React 实现登录页和欢迎页，后端提供登录接口，前后端通过 API 交互。最终通过 Dockerfile 实现一体化部署。

## 背景与价值
- 保护系统资源，防止未授权访问
- 提升用户体验，支持前后端分离架构
- 便于后续扩展用户体系、权限管理
- 一体化部署，便于运维和交付

## 范围
- 前端：React 登录页、欢迎页、API 对接
- 后端：Spring Boot 登录接口，简单用户校验
- 集成：Dockerfile 支持前后端一体化部署

## 不在本次范围
- 第三方 OAuth 登录
- 持久化用户（本次仅内存用户）
- 复杂权限体系
